SZURL: a link-shortening microservice

How to use this service

Send a valid URL as parameter to https://szurl.herokuapp.com/make/.

This service will return a JSON object with the original URL and the shortened URL.

Future visits to the shortened URL will redirect to the original URL.

Example

Input: https://szurl.herokuapp.com/make/http://phys.org/news/2016-10-curious-tilt-sun-undiscovered-planet.html

Output: {"original":"http://phys.org/news/2016-10-curious-tilt-sun-undiscovered-planet.html","shortened":"https://szurl.herokuapp.com/go/23"}

Use: https://szurl.herokuapp.com/go/23
